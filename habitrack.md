# HabiTrack

Tech Stack: XCode, Swift

An app that allows users to track their habits across a week, they would be able to add a habit, check it off and see streaks.

## Features

* Add new behaviors to track.
* See the list of behaviors on home screen.
* Log a behavior
* See a detailed view of a behavior after clicking on it which contains informationm about how many times the habit is logged and when it was last logged.
* See how often each behavior has been logged.
* Delete a behavior
